============================================================

QUICK INSTALLATION (2 MINUTES):
============================================================

METHOD 1: AUTOMATIC INSTALLATION (RECOMMENDED)
------------------------------------------------
1. Right-click on "install.bat"
2. Select "Run as administrator"
3. Click "Yes" on the UAC prompt
4. Follow the installation prompts
5. Installation complete!

METHOD 2: MANUAL INSTALLATION
------------------------------------------------
1. Open Command Prompt as Administrator
2. Navigate to the screen saver directory:
   cd C:\path\to\AtomicClockScreenSaver
3. Run the installation command:
   python AtomicClock.scr /install

METHOD 3: COPY & REGISTER MANUALLY
------------------------------------------------
1. Copy "AtomicClock.scr" to C:\Windows\System32\
2. Open Registry Editor (regedit.exe)
3. Navigate to: HKEY_CURRENT_USER\Control Panel\Desktop
4. Set "SCRNSAVE.EXE" to "C:\Windows\System32\AtomicClock.scr"
5. Set "ScreenSaveTimeOut" to "300" (5 minutes)
6. Set "ScreenSaveActive" to "1"

============================================================

TROUBLESHOOTING:
============================================================

PROBLEM: "Access denied" during installation
SOLUTION: Run install.bat as administrator

PROBLEM: "Python not found"
SOLUTION: Install Python 3.8+ from python.org

PROBLEM: Screen saver not appearing in list
SOLUTION: Verify AtomicClock.scr is in C:\Windows\System32\

PROBLEM: Weather not updating
SOLUTION: Check internet connection and API keys

PROBLEM: Screen saver runs slowly
SOLUTION: Update graphics drivers, close other apps

============================================================

UNINSTALLATION:
============================================================

METHOD 1: AUTOMATIC UNINSTALL
1. Right-click on "uninstall.bat"
2. Select "Run as administrator"
3. Confirm uninstallation

METHOD 2: MANUAL UNINSTALL
1. Delete C:\Windows\System32\AtomicClock.scr
2. Clear registry settings:
   - Delete "SCRNSAVE.EXE" value
   - Set "ScreenSaveActive" to "0"
